for parser_v2.py

параметр exceptions - список символов, которые не должны подсчитываться (задан по умолчанию, но можно изменить)

параметр selected - список нужных символов для подсчёта (считает только их), в случае отсутствия будут считаться
все символы не входящие в exceptions

get_count - наполняет sym_dict ключами(символами) и значениями (количество символов), возвращает sym_dict

get_histogram - считает процентное соотношение всех символов в sym_dict и отображает в цикле символ - его процент

get_histogram:
    vertical - при True отображает гистограмму
    down_top - указывает как будет выглядеть гистограмма (при True будет стандартное отображение в виде гор)
